---
title: Get available shows
description: Retrieve a list of all TV shows available in the API
---

Get a list of all TV shows that have quotes available in the database.

## Endpoint

```
GET /quotes/shows
```

## Response

<ResponseField name="shows" type="string[]" required>
  An array of TV show names available in the database
</ResponseField>

## Example request

```bash curl
curl https://quotes.jepcd.com/quotes/shows
```
